# PhishShield AI

A professional phishing detection web application built with HTML, CSS, and JavaScript.

## Features
- URL and message phishing scan
- Risk score with low, suspicious, and high-risk labels
- Transparent evidence list and recommended next actions
- Local scan history with clear option
- Security awareness and safety checklist pages

## Run
Open `index.html` in a browser. The current version is fully static and stores scan history in local browser storage.
